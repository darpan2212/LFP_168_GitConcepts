1) git init
2) git remote add origin URL
3) git status
4) git add (-A,*,.,fileName)
5) git commit -m "your msg"
6) git pull origin BRANCH_NAME
7) git push origin BRANCH_NAME

Git Branches
1) git branch => to show the list of local branches
2) git branch -M BRANCH_NAME => create and checkout to new branch
3) git checkout BRANCH_NAME => checkout to existing branch
4) git merge ANOTHER_BRANCH_NAME => merge branch with your current active branch
